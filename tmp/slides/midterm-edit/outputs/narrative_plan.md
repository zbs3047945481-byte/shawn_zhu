# 中期报告PPT修改说明

## Audience
毕业设计中期答辩老师与指导教师。

## Objective
在保留原始PPT视觉样式的前提下，增强汇报结构完整性与文字表达清晰度，使其更适合正式中期汇报场景。

## Narrative Arc
封面 -> 目录 -> 本阶段目标与任务 -> 已完成任务 -> 四个关键问题及解决思路 -> 尾页。

## Slide List
1. 封面页
2. 目录页
3. 本阶段工作目标与任务要求
4. 已完成任务情况
5. 问题一：Non-IID数据划分
6. 问题二：即插即用插件化设计
7. 问题三：特征共享中的隐私风险缓解与稳定性增强
8. 问题四：特征蒸馏约束缓解客户端偏离
9. 尾页

## Source Plan
- 现有PPT版式、字体、配色与图示位置
- 中期报告.docx中的“任务要求”“已完成任务”“存在的问题和拟解决方法”正文

## Visual System
完全沿用现有PPT的封面风格、章节页结构、问题页版式和配色，不重建设计语言。

## Asset Plan
保留原始代码截图与示意图，仅通过重新布局扩大显示面积，优先提升后半部分图示可读性。

## Editability Plan
使用可编辑PPT对象直接修改原有文本框内容、复制既有页面生成目录与尾页，并导出新的可编辑PPTX文件。
